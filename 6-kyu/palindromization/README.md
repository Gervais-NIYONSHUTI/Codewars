<h2><a href=https://www.codewars.com/kata/5840107b6fcbf56d2000010b/train/java target="_blank">Palindromization</a></h2><h3>6 kyu</h3><p>You are given string <code>elements</code> and an int <code>n</code>.Your task is to return a string that is a palindrome using elements from the string <code>elements</code> with the length <code>n</code>.</p><h2 id="the-format-of-the-palindromization">The format of the palindromization:</h2><ul><li>Your palindrome begins with the first element of <code>elements</code>.</li><li>After obtaining a pair, you insert the next element in <code>elements</code> to the palindrome.</li><li>The next element will be paired inside the first pair.</li><li>Repeat</li><li>If you have reached the last element of <code>elements</code> then repeat the process from the start.</li></ul><h2 id="error-cases">Error cases:</h2><p>If the string elements is empty or <code>n</code> is smaller than <code>2</code>, return the string <code>"Error!"</code>.</p><h2 id="examples">Examples:</h2><pre><code>For elements = "123" n = 2 =&gt; result = "11"n = 3 =&gt; result = "121"n = 4 =&gt; result = "1221"n = 5 =&gt; result = "12321"n = 6 =&gt; result = "123321"n = 7 =&gt; result = "1231321"n = 8 =&gt; result = "12311321"n = 9 =&gt; result = "123121321"n = 10=&gt; result = "1231221321"</code></pre>
